 2.关系运算符（比较运算符）
   >   <  >= <=          ==    ===
   1》比较运算符 返回的结果是boolean值true false
   2》两个数值会正常比较大小(数学)
   3》如果两个字符串比较大小 
  从第一个开始进行比较 ASCII的大小： 
  相同  继续比较第二个 ... 直到得到结果
  不同  直接返回结果 ，不在比较
   4》一个是字符  一个是数字
     "a">10    字符串纯字母 不能比较大小 直接返回false
      '10'>20  比较大小的时候数据类型不同 隐式数据类型转换
       如果字符串 "aa"  转换出来 NaN
       如果字符串 "10"  转换成对应数值
       如果字符串 "10a" 不能转换 NaN
       如果字符串 ""     转换成 0

     undefined  转换成 NaN
     null       转换成 0   

     true   转换成1
     false   转换成0    

   ==   相等  判断 值是不是相等  
      null==undefined   ture
      两个都是字符串  比较ASCII码
      两个数值       正常比较
      两个引用类型    比较地址是不是相同
   ===  全等 不仅比较值的大小还比较 数据类型
