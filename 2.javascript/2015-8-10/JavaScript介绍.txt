4. JavaScript语法特点
   1.基于对象
   在JavaScript中 一切皆对象
   2.基于事件  
   JavaScript 没有事件   他没有存在的必要
   3.松散型  （弱类型）
   强类型 int a=10;     int a="aaa";
   松散型 var a=1;     var a="str";
   4.解释型
   JavaScript脚本由浏览器去编译执行

   JavaScript脚本  浏览器 支持  浏览器支持JavaScript

   跨平台   脚本只需要写一次
